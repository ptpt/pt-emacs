* Installation
1) Place all these files under ~/.emacs.d/
2) Add the following lines to your ~/.emacs

#+BEGIN_SRC emacs-lisp
   (add-to-path 'load-path "~/.emacs.d/")
   (require 'pt-simple)
#+END_SRC
